# Zero-Cost Passive Income – Quick Start

> A step-by-step execution guide to go from **$0 → first dollar** using only free tools.  
> Follow this exactly. Do not optimize early.

---

## The Rules

1. No paid tools  
2. No custom backend  
3. One income path only  
4. Shipping > perfection  

If you break the rules, you slow yourself down.

---

## Choose ONE Path (5 minutes)

Pick **one** and commit for 7 days.

- **Affiliate Content Site** (fastest, lowest effort)
- **Digital Template / Guide** (highest ROI per hour)
- **Newsletter + Affiliate Stack** (compounding)

For first-time execution, choose **Affiliate Content Site**.
